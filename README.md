# my-todo-project

Todo 관리를 위한 웹 애플리케이션입니다.

## Features

- 편리한 UI로 Todo를 쉽게 생성하고 삭제할 수 있습니다.
- 필터링 기능을 사용하여 Todo를 편하게 볼 수 있습니다.
- yarn start를 사용하여 간편하게 프로젝트를 볼 수 있습니다.
- night 모드를 제공하여 사용자가 좀 더 원하는 밝기의 UI로 TodoList를 볼 수 있습니다.
